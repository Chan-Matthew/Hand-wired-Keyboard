# Build Log of custom hand wired keyboard!!

<b> Materials Used: </b>
<ul>
  <li> Pro Micro Arduino </li>
  <li> Wires </li>
  <li> 1N4148 Diodes </li>
  <li> White blank keycaps from Maxkeyboards </li>
  <li> M2 screws 16mm (they don't even fit the case because the holes are cut slightly too big </li>
</ul>
